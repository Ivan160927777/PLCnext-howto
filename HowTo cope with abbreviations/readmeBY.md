# Як спраўляцца з абрэвіятурамі. #
> [!TIP]
> English version is [here](./readme.md).

## Спіс часцей за ўсё выкарыстаных скарачэнняў. ##

**ARP** - Automation Runtime Platform (Платформа выканання аўтаматызацыі) \
**GDS** - Global Data Space (Глабальны прастор дадзеных) \
**ESM** - Execution and Synchronization Manager (Менеджэр выканання і сінхранізацыі)\
**ACF** - Automation Component Framework (Структура кампанентаў аўтаматызацыі) \
**PLM** - Program Library Manager (Менеджэр бібліятэкі праграм) \
**SM** - Service Manager (Менеджэр па абслугаванню) \
**AXL** - Axioline \
**RSC service** - Remote Service Call service (Паслуга аддаленага выкліку службы падтрымкі)
